## 简介

小众软件论坛发帖或回复时，粘贴、拖曳或上传按钮选择图片/文件，自动上传到 [h1.appinn.me](https://h1.appinn.me/) 并转为对应的 Markdown 格式输出。

## 限制

### 文件大小

20 MB

### 文件类型

- 图片
- 视频
- 音频
- pdf
